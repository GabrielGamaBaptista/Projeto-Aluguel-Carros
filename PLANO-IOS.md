
Plano iOS — AluguelCarrosApp                                                                                                                                                                                    │
│                                                                                                                                                                                                                 │
│ Contexto e Motivacao                                                                                                                                                                                            │
│                                                                                                                                                                                                                 │
│ O app AluguelCarrosApp roda apenas em Android. O objetivo e instalar em 1 iPhone pessoal, desenvolvendo no Windows, com custo zero (sem Apple Developer Program de $99/ano). Migracao futura para conta paga e  │
│ possivel sem mudancas de codigo.                                                                                                                                                                                │
│                                                                                                                                                                                                                 │
│ Analise do Plano Original (ChatGPT) — Falha Critica Identificada                                                                                                                                                │
│                                                                                                                                                                                                                 │
│ O plano original propunha: Codemagic builda IPA → AltStore instala no iPhone.                                                                                                                                   │
│                                                                                                                                                                                                                 │
│ Problema: Codemagic nao consegue gerar IPA instalavel com Apple ID gratuito. Code signing para dispositivo fisico requer:                                                                                       │
│ - Conta Apple Developer paga ($99/ano), OU                                                                                                                                                                      │
│ - Build local no Xcode (Mac) com free provisioning                                                                                                                                                              │
│                                                                                                                                                                                                                 │
│ Solucao encontrada: Codemagic gera IPA unsigned (sem assinatura) → SideStore no iPhone re-assina com Apple ID gratuito e instala.                                                                               │
│                                                                                                                                                                                                                 │
│ Abordagem Escolhida: $0/ano com SideStore                                                                                                                                                                       │
│                                                                                                                                                                                                                 │
│ ┌────────────────────┬────────────────────────────────────────────────────────────────────────────────────────────────────────────┐                                                                             │
│ │      Aspecto       │                                                  Detalhe                                                   │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │
│ │ Build              │ Codemagic free tier (500 min/mes macOS M2, ~20-30 builds/mes)                                              │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │
│ │ Distribuicao       │ SideStore no iPhone (re-assina IPA unsigned)                                                               │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │
│ │ Expiracao          │ 7 dias (SideStore renova automaticamente via VPN local no iPhone)                                          │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │
│ │ Push Notifications │ NAO funcionam (APNs requer certificado da conta paga). Locatario iOS ve notificacoes apenas ao abrir o app │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │
│ │ Limite de apps     │ 3 apps sideloaded por Apple ID (SideStore conta como 1)                                                    │                                                                             │
│ ├────────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤                                                                             │

│ 1.1 Registrar app iOS no Firebase Console                                                                                                                                                                       │
│                                                                                                                                                                                                                 │
│ 1. Abrir console.firebase.google.com                                                                                                                                                                            │
│ 2. Selecionar projeto aluguel-carros-30b83                                                                                                                                                                      │
│ 3. Clicar na engrenagem ao lado de "Project Overview" → "Project settings"                                                                                                                                      │
│ 4. Rolar ate secao "Your apps"                                                                                                                                                                                  │
│ 5. Clicar "Add app" → icone da Apple (iOS+)                                                                                                                                                                     │
│ 6. Preencher:                                                                                                                                                                                                   │
│   - Apple bundle ID: com.gabrielgama.aluguelcarrosapp                                                                                                                                                           │
│   - App nickname: AluguelCarrosApp iOS                                                                                                                                                                          │
│   - App Store ID: deixar vazio                                                                                                                                                                                  │
│ 7. Clicar "Register app"                                                                                                                                                                                        │
│ 8. Clicar "Download GoogleService-Info.plist" → salvar                                                                                                                                                          │
│ 9. Colocar o arquivo em:                                                                                                                                                                                        │
│ C:\Projetos\AluguelCarrosApp\ios\AluguelCarrosApp\GoogleService-Info.plist                                                                                                                                      │
│ 9. (Este caminho ja esta no .gitignore — nao sera commitado)                                                                                                                                                    │
│ 10. Abrir o arquivo .plist com um editor de texto (Notepad, VS Code)                                                                                                                                            │
│ 11. Procurar a chave REVERSED_CLIENT_ID — o valor se parece com:                                                                                                                                                │
│ com.googleusercontent.apps.254869962668-xxxxxxxxxxxxxxxxxxxx                                                                                                                                                    │
│ 12. Anotar esse valor — sera usado no Passo 2.3 (Info.plist URL scheme)                                                                                                                                         │
